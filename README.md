# actually.mi

A REST API server built with Express and TypeScript.

## Tech Stack

- Node.js / Express
- TypeScript
- Prisma ORM
- PostgreSQL
- JWT Authentication
- bcrypt

## Getting Started

### Prerequisites
- Node.js 18+
- PostgreSQL

### Install dependencies
npm install

### Run in development
npm run start:dev

### Build for production
npm run start:prod

### Run Jest tests
npm run test

### Run Jest tests with code coverage
npm run test:coverage
